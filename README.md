# Examen Eventos y Recordatorios

Este proyecto es una aplicación de Android para gestionar tareas pendientes y completadas. La aplicación permite a los usuarios agregar, ver y marcar tareas como completadas.

## Características

- Agregar nuevas tareas
- Ver tareas pendientes
- Ver tareas completadas
- Marcar tareas como completadas
- Eliminar tareas

## Tecnologías Utilizadas

- Kotlin
- SQLite
- Android Studio

## Estructura del Proyecto

- `MainActivity.kt`: Actividad principal donde se pueden agregar y ver tareas.
- `PendingTasksActivity.kt`: Actividad para ver las tareas pendientes.
- `DoneTasksActivity.kt`: Actividad para ver las tareas completadas.
- `DatabaseHelper.kt`: Clase para gestionar la base de datos SQLite.
- `Task.kt`: Clase de datos para representar una tarea.

## Instalación

1. Clona este repositorio:
    ```sh
    git clone https://github.com/jacobgalactico/ExamenEventosRecordatorios.git
    ```

2. Abre el proyecto en Android Studio.

3. Compila y ejecuta la aplicación en un dispositivo o emulador Android.

## Uso

1. Abre la aplicación.
2. Agrega una nueva tarea escribiendo en el campo de texto y presionando el botón "Agregar Tarea".
3. Usa los botones "Mostrar Pendientes" y "Mostrar Completadas" para alternar entre las vistas de tareas.
4. Marca una tarea como completada tocando en la tarea en la lista de tareas pendientes.
5. Elimina una tarea manteniendo presionada la tarea en la lista.
